# Quant Algorithm

An algorithmic trading bot built in Python, featuring automated trade execution, backtesting, and data analysis.

## Features
- Automated trade execution via bot.py
- Backtesting engine to validate strategies
- Market data fetching and processing
- Trade logging and performance tracking

## Project Structure
```
├── bot.py          # Main trading bot
├── backtest.py     # Backtesting engine
├── data.py         # Data fetching & processing
├── config.py       # API keys & settings (not tracked)
└── .gitignore
```

## Setup

1. Clone the repo
```bash
git clone https://github.com/RG1579/quant-alg.git
cd quant-alg
```

2. Install dependencies
```bash
pip install -r requirements.txt
```

3. Add your API keys to `config.py`

4. Run the bot
```bash
python bot.py
```

## Backtesting
```bash
python backtest.py
```
